# Styling Rules

- All colors via CSS custom properties — never use arbitrary Tailwind values (`[#hex]`, `bg-red-500`)
- Use design tokens: `bg-primary`, `text-muted-foreground`, `border-accent-1`, etc.
- Domain-to-token mapping lives in `features/canvas/theme.ts` — not in CSS or components
- Use CVA variants on Button/Badge — don't override with className
- Inline styles ONLY for React Flow handles (they require hex values)
- Never edit `src/components/ui/` files — they are auto-generated by shadcn. Customize via CVA variants or create wrapper components
- New shadcn components: `pnpm dlx shadcn@latest add <component>`
